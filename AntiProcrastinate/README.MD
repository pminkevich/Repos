Anti Procrastine
--------------------------------------------------------------------------

La idea de este programa surgió porque mi sobrina pasaba mucho tiempo viendo
videos tontos por así decirlo
--------------------------------------------------------------------------
Anti Procrastine te permite regular por medio de un panel administrador el tiempo
que quieres que alguien mire videos de ocio y videos de Aprendizaje. De esta manera
te aseguras que no siempre se este mirando videos para ocio y cada cierto tiempo 
que tu puedes configurar se muestre en pantalla un video para aprender.

(Entity Framework, API YOUTUBE)
